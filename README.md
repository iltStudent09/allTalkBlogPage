## Design Rationale

The design of AllTalk Blog is guided by the principles of clarity, accessibility, and maintainability. The site uses a clean, modern layout with a focus on readability and ease of navigation. The color palette was chosen to provide strong contrast between content and background, ensuring that text is easy to read for all users. The use of a serif font for body text and sans-serif for navigation and headings creates a visual hierarchy that helps users quickly scan and understand the structure of each page.

Flexbox is employed throughout the layout to provide flexible, robust alignment and spacing, making it easy to adapt the design to different screen sizes. Responsive breakpoints are implemented for mobile, tablet, and desktop, ensuring that the site remains usable and visually appealing on any device. The navigation bar is always visible and consistent across all pages, allowing users to move between sections without confusion.

Each page is organized into logical sections: header, navigation, main content, and footer. This makes the codebase easy to extend and maintain. CSS is modularized, with separate files for each main page, so that styles can be updated independently. The contact form is simple and accessible, using clear labels and large touch targets for mobile users. Overall, the design aims to balance aesthetics with usability, providing a welcoming experience for both readers and contributors.
# AllTalk Blog Page

AllTalk Blog is a simple, responsive blog website built with HTML and CSS. It demonstrates clean layout, navigation, and basic responsive design using Flexbox and media queries.

## Features
- Home, About, and Contact pages
- Responsive design for mobile, tablet, and desktop
- Clean navigation and consistent styling

## Project Structure

- `index.html`: Main blog homepage
- `styles.css`: Main site-wide styles
- `pages/contact/contact.html`: Contact page
- `pages/contact/contact.css`: Contact page styles
- `pages/about/about.html`: About page
- `pages/about/about.css`: About page styles

## Usage
Open `index.html` in your browser to view the blog. Use the navigation links to access the About and Contact pages.

## Customization
You can add more posts, pages, or adjust the styles as needed. All pages are static and require no backend.
